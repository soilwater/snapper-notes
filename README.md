## How to Use

1. **Capture**
   - Tap **Capture** and take a photo.
   - GPS location and timestamp are recorded automatically.

2. **Add Notes**
   - Write a short comment.
   - Add up to three keywords (type and press Enter).

3. **Save**
   - Your capture is stored locally on your device.

4. **Review & Export**
   - Browse all saved captures.
   - Download individual images with metadata.
   - Export all data as a CSV file.
   - Delete individual captures or clear everything.

## Installation

### Use in a Browser
- Open `index.html` in any modern browser.
- No build step, no dependencies.

### Install on Mobile (Recommended)

**iOS (Safari)**
1. Open the app
2. Tap **Share**
3. Select **Add to Home Screen**

**Android (Chrome)**
1. Open the app
2. Tap the menu
3. Select **Add to Home screen**
